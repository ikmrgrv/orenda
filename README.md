# Orenda

For DEMO purpose only.
Project is in `alpha` phase. Please do not use it for `production` use.

## Project setup

```js
npm i orenda
```

## Basic Usage

```js
<template>
  <OrendaButton
    background="#fff"
    color="#000"
    text="Yo! Button works"
  />
</template>

<script>
import { OrendaButton } from 'orenda'

export default {
  components: {
    OrendaButton
  }
}
</script>
```
